Создание группы:
CREATE GROUP kpi WHERE id = "kpi_live"
Под kpi следует понимать псевдоним, который привязан к идентификатору группы public23762795 либо краткому имени kpi_live.

Удаление группы:
DROP GROUP kpi
Удаляет объектное представление группы из памяти по ее псевдониму.

Данные группы:
SHOW GROUP kpi
Текстовое представление основных данных о группе по ее псевдониму.
