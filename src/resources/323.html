Выбор количества лайков для соответствующего поста:
SELECT POST FROM kpi BY likes
Под kpi следует понимать псевдоним группы.

Выбор количества лайков для соответствующего поста в заданном интервале:
SELECT POST FROM kpi START 1 TO 999 BY likes
SELECT POST FROM kpi START 1 TO ? BY likes
Под kpi следует понимать псевдоним группы, выбор в диапазоне постов от 1 до 999.
Нумерация постов начинается с 0. Знак вопроса указывает на общее количество постов группы

Выбор количества комметариев для соответствующего поста:
SELECT POST FROM kpi START 1 TO 999 BY comment
SELECT POST FROM kpi START 1 TO ? BY comment
Под kpi следует понимать псевдоним группы, выбор в диапазоне постов от 1 до 999.
Нумерация постов начинается с 0. Знак вопроса указывает на общее количество постов группы

Выбор соотношения комментариев к лайкам в заданном интервале для соответствующего поста:
SELECT POST FROM kpi START 1 TO 999 BY comment/likes
SELECT POST FROM kpi START 1 TO ? BY comment/likes
Под kpi следует понимать псевдоним группы, результат - отношение комментариев к лайкам(число с плавающей точкой), для постов в диапазоне постов от 1 до 999.
Нумерация постов начинается с 0. Знак вопроса указывает на общее количество постов группы
